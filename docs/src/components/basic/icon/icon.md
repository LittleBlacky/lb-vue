# Icon 图标

## 安装

首先需要安装 `@vicons/ionicons5` 图标库：

```bash
pnpm add @vicons/ionicons5
```

## 基本用法

<preview path="./demos/BasicUsage.vue"></preview>

## 属性

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| color  | string | #000 | 图标颜色 |
| size   | number\|string | 1rem | 图标大小 |
| spin   | boolean | false | 是否旋转 |
